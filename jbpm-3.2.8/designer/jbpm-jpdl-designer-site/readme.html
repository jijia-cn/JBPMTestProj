JBoss jBPM GPD Update Site Installation

To install the designer you have to work your way through a number of very easy steps. The described installation process has been tested with the 3.4 version of Eclipse.

  • Start Eclipse and use the regular plugin installation mechanism : select 'Help->Software Updates...'
  • On the 'Software Updates and Add-ons' dialog, navigate to the 'Available Software' tab and push the 'Add Site...' button.
  • On the 'Add Site' dialog, push the 'Archive...' button unless you have extracted the archive containing this readme file. In that case push the 'Local...' button.
  • On the following selection dialog, navigate to the archive containing this readme file in case you did not extract or to the folder into which you did the extraction in the other case.
  • The 'jBPM jPDL Updates' category shows up. Select the category and push the 'Install...' button.
  • After having resolved the dependencies needed, Eclipse shows you the 'Install' wizard. Select 'Next >' and on the following page make sure you have selected the 'I accept the terms of the license agreement' option to enable the 'Finish' button.
  • Select the 'Finish' button and when Eclipse asks you, restart.

You're done!
